# 打包清单 - 食材养生公众号文章生成工具

## 需要打包的文件/文件夹

```
📦 养生工具.zip
├── 📄 README.md              # 快速开始指南
├── 📄 使用教程.txt           # 详细使用说明
├── 📄 双击运行.bat           # 一键启动脚本
├── 📄 run.js                 # 主程序
├── 📄 package.json           # 项目配置
├── 📄 package-lock.json      # 依赖锁定
├── 📄 config.json            # API 配置（需要用户自行填写密钥）
│
├── 📁 lib/                   # 核心功能模块
│   ├── ai.js
│   ├── topic-generator.js
│   ├── article-generator.js
│   ├── image-generator.js

│

│

│   ├── generate.py
│   └── requirements.txt
│
└── 📁 outputs/               # 输出目录（保留空文件夹）
    └── .gitkeep
```

## 不需要打包的文件

- ❌ `.git/` - Git 版本控制目录
- ❌ `.worktrees/` - Git 工作树
- ❌ `node_modules/` - 依赖目录（用户首次运行会自动安装）
- ❌ `test/` - 测试文件（小白用户不需要）
- ❌ `docs/` - 文档目录（如果有）
- ❌ `.gitignore` - Git 忽略配置
- ❌ `CLAUDE.md` - 开发文档

## 打包步骤

### 方法一：使用 7-Zip 打包

1. 选中需要打包的文件和文件夹（按上面清单）
2. 右键 → 7-Zip → 添加到压缩包...
3. 格式选择：zip
4. 压缩等级：标准
5. 确定

### 方法二：使用命令行

```bash
# Windows PowerShell
Compress-Archive -Path "README.md","使用教程.txt","双击运行.bat","run.js","package.json","package-lock.json","config.json","lib","prompts","outputs" -DestinationPath "养生工具.zip"
```

## 注意事项

1. **不要打包 node_modules**：体积太大（几百MB），且用户环境可能不同
2. **保留 config.json**：但要提醒用户修改自己的密钥
3. **outputs 文件夹**：保留空文件夹，让用户知道输出位置
4. **Python 依赖**：zimage-skill 需要 requests 和 Pillow，用户首次使用需要自行安装
